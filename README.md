# 🗂️ Portfólio — Gabriel Borges

Meu portfólio pessoal, desenvolvido com foco em performance, 
design minimalista e experiência limpa para recrutadores.

🔗 **[bielborgesc.vercel.app](https://bielborgesc.vercel.app)**

## Stack

- React 19 + Vite
- TypeScript
- Tailwind CSS
- Deploy automático via Vercel

## Funcionalidades

- Versão em PT e EN
- Currículo para download
- Dark mode
- Totalmente responsivo

## Rodando localmente

git clone https://github.com/bielborgesc/portifolio  
cd portifolio  
npm install  
npm run dev
